# 🩸 Blood Bank Management System

A web-based **Blood Bank Management System** built with **PHP, MySQL, HTML, CSS, and JavaScript**.  
This project helps manage blood donors, requests, and camps, making it easier for hospitals and individuals to find and donate blood.

---

## 🚀 Features

- 🔐 **Admin Dashboard** – Manage donors, requests, blood camps, and reports.  
- 🧑‍🤝‍🧑 **Donor Registration & Login** – Donors can register, log in, and update their details.  
- 🩸 **Blood Requests** – Users can request blood by blood group and location.  
- 📅 **Blood Donation Camps** – Display and manage upcoming donation camps.  
- 📊 **Reports** – Generate reports for admin use.  
- 📱 **Responsive UI** – User-friendly design for easy access.  

---

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** PHP  
- **Database:** MySQL  
- **File Storage:** Git LFS (for large files like `.sql` and `.zip`)
